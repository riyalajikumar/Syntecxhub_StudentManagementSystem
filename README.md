# Student Management System

## Project Overview

The Student Management System is a Python-based Command Line Interface (CLI) application developed to manage student records efficiently. The system allows users to add, update, delete, and view student information while maintaining data permanently using JSON file storage.

This project demonstrates Object-Oriented Programming (OOP), file handling, data validation, and CRUD operations in Python.

---

## Features

* Add Student Records
* Update Existing Student Details
* Delete Student Records
* View All Student Records
* Unique Student ID Validation
* JSON File Storage
* Menu-Driven Interface
* Object-Oriented Programming Implementation

